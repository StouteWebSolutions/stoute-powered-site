== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/` directory, or install the plugin through the WordPress plugins screen directly.
1. Activate the plugin through the 'Plugins' screen in WordPress
1. Use the *Tools > Hotjar for WordPress* page to add your Hotjar SiteID for the tracking code.

== Frequently Asked Questions ==

= I'm not a Stoute.co client. Can I still use this plugin? =

Yes, of course! You don't need to be a Stoute.co client to use this plugin.
